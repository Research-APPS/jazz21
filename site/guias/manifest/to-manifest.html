to_manifest

Combina descripción de acorde + resolución de guitarra para una lista de símbolos. Pensado para publicadores, SEO/GEO y bridges hacia apps como CHORDIA.

API: jazz21.to_manifest(symbols: list[str]) → list[dict]. Cada entrada tiene symbol, chord, guitar o error si no se reconoce.
