Importar y usar jazz21

La API de alto nivel vive en el paquete raíz. También puedes importar submódulos (jazz21.notation, jazz21.symbols, jazz21.guitar).

Concepto Python: import jazz21 carga el paquete. Luego accedes a funciones con punto: jazz21.describe_chord(...). print() envía texto a la salida estándar.
