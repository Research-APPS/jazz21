Entorno virtual y pip

jazz21 es una librería Python. Lo habitual es instalarla en un entorno virtual aislado del sistema, para no mezclar dependencias con otros proyectos.

Concepto Python: un venv es una carpeta con su propio python y pip. Activas el entorno en la terminal y todo lo que instales con pip queda ahí dentro.

En tu máquina (fuera del navegador):

python3.10 -m venv .venv
source .venv/bin/activate          # macOS / Linux
# .venv\Scripts\activate           # Windows

python -m pip install -U pip setuptools wheel
pip install -e ".[dev]"            # desde el clon del repo

Comprueba que jazz21 responde:

python -c "import jazz21; print(jazz21.__version__)"
pytest -q

En esta biblioteca web, el código jazz21 corre con Pyodide (Python en WASM). No necesitas venv en el navegador, pero sí para desarrollo local serio.
