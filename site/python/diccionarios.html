Leer un diccionario de acorde

describe_chord devuelve un dict con metadatos del símbolo: forma canónica, intervalos, figura music21, etc.

Concepto Python: un dict es un mapa clave → valor. Accedes con corchetes: resultado["canonical"]. Si la clave no existe, Python lanza KeyError — usa .get("clave") para un valor por defecto.
